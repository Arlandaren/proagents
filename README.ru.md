**proagents** — это курируемая библиотека без лишних префиксов, содержащая **более 770 профессиональных системных промптов и рабочих процессов (workflows)** для разработки с использованием ИИ (Cursor, Claude Code, Windsurf, Zed и кастомные LLM-агенты). Структурированная по функциональным доменам, эта библиотека создана для замены стандартных, некачественных ответов ИИ («AI slop») кодом промышленного уровня, выдающимся визуальным стилем и строгим соблюдением требований безопасности.

---

## ⚡ Момент прозрения (Aha! Moment): Обычный ИИ против proagents

Большинство LLM пишут обобщенный код, похожий на базовые туториалы. **proagents** заставляет модель действовать как опытный специалист со строгими правилами.

| Критерий | Стандартный вывод ИИ ❌ | Вывод с использованием proagents |
| :--- | :--- | :--- |
| **Визуальный стиль** | Стандартные серые/синие кнопки Tailwind, скучная разметка, шрифты по умолчанию. | Уникальные HSL-палитры, плавные микроанимации, стеклянный интерфейс (glassmorphism), премиальная типографика. |
| **Безопасность кода** | Захардкоженные секреты, отсутствие обработки ошибок, игнорирование крайних случаев. | Никакого хардкода, строгая валидация входящих данных, обработка исключений с конкретными типами ошибок. |
| **Логика и Тестирование** | «Сначала пишем код, тесты когда-нибудь потом». | Строгие циклы TDD, четкое разделение по папкам, явные контракты интерфейсов. |
| **Производительность** | Избыточные ререндеры, неоптимизированные медиа-активы. | Точечные операции с DOM, физика пружин (spring physics), оптимизация LCP. |

## 🛠️ Интерактивный CLI-инструмент

Вместо того чтобы вручную искать файлы в сотнях папок, используйте легковесный скрипт `proagents.py` без внешних зависимостей для поиска и установки правил.

```bash
# 1. Поиск правил по ключевому слову (например, React)
python proagents.py search react

# 2. Получить описание и метаданные конкретного правила
python proagents.py install react-patterns --info

# 3. Установить правило напрямую в конфигурацию Cursor
python proagents.py install react-patterns --cursor

# 4. Или вывести в stdout для добавления в CLAUDE.md
python proagents.py install production-deployment --stdout >> CLAUDE.md
```

## 📂 Структура репозитория

Все ресурсы организованы в четыре функциональные директории:

```
proagents/
├── proagents.py                # Легковесный CLI-менеджер
├── rules/                      # Глобальные правила и дизайн-системы
│   ├── core/                   # Гигиена кода, требования к git, обработка ошибок
│   └── taste/                  # Теория цвета HSL, кастомная анимация и сетки разметки
├── personas/                   # Системные инструкции для специализированных ролей
│   ├── engineering/            # 3D Metal разработчик, разработчик смарт-контрактов, SRE
│   ├── design/                 # UX-исследователь, хранитель бренда, моушн-дизайнер
│   ├── operations/             # Аудитор кода, релиз-менеджер, скрам-мастер
│   ├── business-strategy/      # Разработчик GTM, составитель предложений, стартап-ментор
│   └── specialized/            # Эксперты в узких нишах (юристы, переводчики, аналитики)
├── workflows/                  # Пошаговые чек-листы выполнения задач
│   ├── development/            # Циклы TDD, гайды по развертыванию, чек-листы релизов
│   ├── qa-testing/             # Аудит доступности WCAG, тестирование адаптивности, e2e-сценарии
│   ├── security-compliance/    # Моделирование угроз OWASP, Dependabot, аудит SOC2
│   └── business-ops/           # Сортировка задач на GitHub, отчеты по операциям
└── templates/                  # Шаблоны разметки, спецификаций и архитектурных решений (ADR)
```

## ⚙️ Как интегрировать

### 🎯 Cursor
Установите любой шаблон напрямую в корень вашего проекта:
```bash
python proagents.py install react-patterns --cursor
```
Это создаст файл `.cursorrules` со строгими инструкциями для Cursor AI.

### 🤖 Claude Code
Добавьте конкретный воркфлоу или чек-лист релиза прямо в ваш `CLAUDE.md`:
```bash
python proagents.py install launch-strategy --stdout >> CLAUDE.md
```

### 💻 Zed / Windsurf / Trae
Выведите текст любого промпта в консоль и вставьте его в настройки системных инструкций вашего редактора:
```bash
python proagents.py install ux-architect --stdout
```
